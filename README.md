# 📊 Bank Customer Churn Analysis (Power BI)

## 📌 Project Overview
This project analyzes **customer churn behavior** in a banking dataset using **Power BI**.  
It aims to identify key churn drivers and help businesses improve **customer retention strategies** through data-driven insights.

---

## 🎯 Business Problem
Customer churn directly impacts revenue and long-term growth.

This project answers:
- Why are customers leaving?
- Which customer segments are at higher risk?
- What factors influence churn the most?

---

## 🛠️ Tools & Technologies
- **Power BI** – Dashboard creation & visualization  
- **DAX (Data Analysis Expressions)** – KPI calculations  
- **Power Query** – Data cleaning & transformation  
- **Data Modeling** – Star schema (Fact & Dimension tables)  

---

## 📂 Dataset Description
The dataset includes:

- Customer ID  
- Age, Gender  
- Geography (France, Germany, Spain)  
- Credit Score  
- Account Balance  
- Tenure  
- Estimated Salary  
- Active Member Status  
- Churn Status (Exited / Retained)  

---

## 📊 Key KPIs (from Dashboard)

- 👥 **Total Customers:** 9,981  
- ❌ **Exited Customers:** 2,036  
- 📉 **Churn Rate:** ~20%  
- ✅ **Retention Rate:** 79.60%  
- 🔄 **Active Members:** 51.50%  

---

## 📈 Dashboard Features

### 🔹 Churn Overview
- Displays total customers, churn count, and retention rate

### 🔹 Customer Segmentation
- Age group analysis (Young, Middle Age, Senior)
- Gender distribution (Male vs Female)

### 🔹 🌍 Geography Analysis
- France has the highest number of customers (~5000+)
- Germany and Spain show lower distribution

### 🔹 📊 Time-Based Analysis
- Churn trends across **Year, Quarter, Month, Day**

### 🔹 🎛️ Interactive Filters
- Gender filter  
- Active vs Inactive members  
- Geography selection  
- Churn status (Exited / Retained)  

---

## 🔍 Key Insights

- 📉 **Inactive customers are more likely to churn**
- ⏳ **Customers with shorter tenure show higher exit rates**
- 🌍 **Churn varies across regions (Germany shows higher risk)**
- 💳 **Customers with lower balances are more likely to leave**
- 👥 **Middle-aged customers form the largest segment**

---

## 🧩 Data Model

This project uses a **Star Schema**:

- **Fact Table**
  - Customer churn metrics and transactional data  

- **Dimension Tables**
  - Customer Details  
  - Geography  
  - Account Information

## 💡 Business Recommendations

- 🎯 Target **inactive customers** with re-engagement campaigns  
- 🆕 Improve onboarding for **new customers (low tenure)**  
- 🌍 Focus retention strategies on **high-churn regions**  
- 💰 Offer benefits for **low-balance customers**
