🎯 Business Problem

Customer churn directly impacts revenue and growth.
This project answers:

Why are customers leaving?
Which segments are most at risk?
What factors influence churn the most?
🛠️ Tools & Technologies
Power BI – Data visualization & dashboarding
DAX (Data Analysis Expressions) – KPI calculations
Data Modeling – Star schema (Fact & Dimension tables)
Data Cleaning & Transformation – Power Query
📂 Dataset Description

The dataset contains customer-level information, including:

Customer ID
Demographics (Age, Gender, Geography)
Financial Data (Balance, Salary, Credit Score)
Account Details (Tenure, Activity Status)
Churn Status (Exited / Retained)
📊 Key KPIs
📌 Total Customers
📌 Churn Rate (%)
📌 Active vs Inactive Customers
📌 Average Account Balance
📌 Customer Distribution by Geography
📈 Dashboard Features
📊 Churn Overview Dashboard – High-level KPIs & trends
👥 Customer Segmentation – Analysis by age, gender, geography
🌍 Geographical Insights – Region-wise churn comparison
💰 Financial Analysis – Balance, salary, and credit impact
🎛️ Interactive Filters & Slicers – Dynamic data exploration
🔍 Key Insights
Customers with low balance & inactive accounts show higher churn probability
Short-tenure customers are more likely to leave early
Certain regions have significantly higher churn rates
Customers with higher credit scores are more stable
🧩 Data Model

The project follows a Star Schema:

Fact Table
Customer Churn Data (transactions & metrics)
Dimension Tables
Customer Details
Geography
Account Information
